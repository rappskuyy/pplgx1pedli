Website **Rombel PPLG X-1** adalah platform digital yang dibuat untuk mempermudah siswa dalam mengakses informasi kelas, mengelola tugas, berkomunikasi, dan menyimpan dokumentasi kegiatan kelas.
Website ini juga dilengkapi dengan **AI Assistant** untuk membantu siswa mencari informasi atau membantu proses belajar.

---

## ✨ Fitur Utama

🔐 **Authentication**
Siswa harus login untuk mengakses beberapa fitur seperti menambahkan informasi siswa.

👨‍💼 **Admin Panel**
Admin dapat menambahkan dan mengelola **tugas kelas**.

🤖 **AI Assistant**
Membantu siswa menjawab pertanyaan dan mencari informasi pembelajaran.

💬 **Chat Kelas**
Siswa dapat berdiskusi dan berkomunikasi secara online.

🖼️ **Gallery**
Menampilkan dokumentasi kegiatan kelas.

👥 **Info Siswa**
Siswa dapat menambahkan atau memperbarui profil mereka setelah login.

📅 **Jadwal Pelajaran**
Menampilkan jadwal pelajaran kelas PPLG X-1.

💰 **Infaq Kelas**
Fitur untuk mencatat dan menampilkan data infaq kelas.
